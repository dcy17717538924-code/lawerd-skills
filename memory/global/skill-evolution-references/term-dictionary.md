# 用词词典

> 所有 skill 的 SKILL.md、steps/*.md、references/*.md 必须遵循本词典。
> 违反本词典的用词将被 `scan-compliance` 脚本标记。

## 一、计算机术语（精确表述，无歧义）

| 术语 | 定义 | 写法 |
|------|------|------|
| `run_skill("name")` | 调用 skill 执行 | 固定写法，不可缩写或变体 |
| `read_file("path")` | 读取文件 | 固定写法，路径以 repo-root 为基准 |
| `IF` / `THEN` / `ELSE` | 条件分支 | 大写，一个条件一行 |
| `AND` / `OR` / `NOT` | 逻辑运算符 | 大写 |
| 前置条件 | 步骤执行前必须满足的条件 | 用 `## 前置条件` 章节 |
| 异常处理 | 失败时的回退路径 | 用 `## 异常处理` 章节，格式 `IF 条件 THEN 动作` |
| 降级 | 主路径失败后的备选方案 | 动词：降级到 |
| 兜底 | 所有已知路径均失败后的最终处理 | 动词：兜底规则 |
| 硬Hook | 等待律师回复，不回复不前进 | 标准写法：`**硬Hook**：` |
| 软Hook | 提醒一次，未回复则跳过 | 标准写法：`**软Hook**：` |
| `run_skill("markitdown")` | 文件格式转换：PDF/Docx/PPT/Excel/HTML/CSV/JSON/EPUB → Markdown | — |
| `run_skill("mineru-ocr")` | 精准解析：扫描件/图片/复杂版式 PDF → Markdown | vlm 模式 + 强制 OCR |
| `openai-whisper` + `medium.pt` | 本地语音转文字 | 模型路径：`{{HOME_WIN}}\.cache\whisper\medium.pt` |

## 二、法律术语（精确区分）

| 术语 | 定义 | 易混淆项 |
|------|------|---------|
| 原告 | 提起诉讼的一方 | — |
| 被告 | 被起诉的一方 | 勿与"被申请人"混淆 |
| 上诉人 | 提起上诉的一方 | — |
| 被上诉人 | 上诉案件中的对方 | — |
| 申请人 | 提出申请的一方（如保全/执行） | — |
| 被申请人 | 被申请的一方 | 勿与"被告"混淆 |
| 案由 | 案件法律关系的分类 | 勿与"诉请"混淆 |
| 诉讼请求 | 原告向法院提出的具体请求 | 勿与"诉讼标的"混淆 |
| 诉讼标的 | 争议的法律关系或财产金额 | — |
| 代理词 | 代理人对案件的综合论述 | 勿与"辩论意见"混淆 |
| 辩论意见 | 针对争议焦点发表的论证 | 勿与"代理词"混淆 |
| 质证意见 | 对对方证据的真实性/合法性/关联性意见 | 勿与"证据目录"混淆 |
| 证据目录 | 己方证据的编号排列 | 勿与"质证意见"混淆 |
| 答辩状 | 被告/被上诉人对起诉/上诉的回应 | — |
| 起诉状 | 原告向法院提出的诉讼文书 | — |
| 上诉状 | 对一审判决不服的救济文书 | — |
| 立案 | 法院受理案件 | — |
| 保全 | 对财产/证据/行为的强制措施 | 分为财产保全/证据保全/行为保全 |
| 管辖权异议 | 对法院管辖权的抗辩 | — |

## 三、禁用词 → 替换词

| 禁用词 | 原因 | 替换为 |
|--------|------|--------|
| "看看" | 口语化歧义 | `read_file()` |
| "看一下" | 口语化歧义 | `read_file()` |
| "打开" | 宾语歧义（文件/进程） | `read_file()` |
| "调起" / "触发技能" / "走 XX skill" | 非标准表述 | `run_skill("name")` |
| "首先 / 其次 / 再次 / 然后 / 接着" | 口语化序列 | 数字序号 1. 2. 3. |
| "我们 / 咱们" | 含混主体 | 具体化：agent / 律师 / {{USER_SHORT_NAME}} |
| "如果...那么..." | 口语化条件 | `IF...THEN...` |
| "若是...则..." | 文言化条件 | `IF...THEN...` |
| "该 XXX"（当 XXX 有确定名称时） | 模糊指代 | 使用确定名称 |
| "相关/相应/有关" | 歧义修饰 | 删除或用确定名称替代 |
| "一般来说" / "通常" | 模糊判断 | `IF...THEN...` 或删除 |
| "相当于" / "类似于" | 模糊类比 | 精确描述 |
| "应该是" | 推测语气 | 确认后陈述，或以 `IF` 分支覆盖 |
| 反问句 | 非指令语气 | 陈述句或 `IF` 分支 |
| `{{VARIABLE}}` | 运行时未展开 | 用确定路径或参数占位符 `[path]` |
| `.../` | 路径省略 | 完整路径或 repo-relative |
| "～"（波浪号） | 模糊范围 | 精确数值或删除 |
| "等"（列举末尾） | 未穷举 | 删除或补全列表 |
