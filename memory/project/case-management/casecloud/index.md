# 案件云助手

## 功能入口

需要操作案件云（查询案件列表和详情、更新案件进度和状态、管理日程和办案记录、查看和更新客户信息、查询财务记录和应收款）时进入本流程。

本功能委托 `run_skill("law086")` 完整执行。本文件仅定义触发条件和参数说明。

## 前置条件

- 律师明确需要操作案件云
- `run_skill("law086")` 可正常调用

## 执行

1. 加载 `errors.md`，注入上下文避免重复踩坑
2. 识别律师意图，提取对应操作参数：
   - 查询案件列表 → 参数：当事人名称 / 案由 / 案号
   - 查询案件详情 → 参数：案号
   - 更新案件进度 → 参数：案号 + 新状态
   - 管理日程 → 参数：案号 + 日期 + 事项
   - 查询客户信息 → 参数：客户名称
3. 委托 `run_skill("law086")`，传入识别后的参数
4. 输出执行结果

## 异常处理

- `IF` 参数无法从对话中提取 `THEN` 追问律师补充必填参数
- `IF` `run_skill("law086")` 返回错误 `THEN` 输出错误详情，询问是否重试
