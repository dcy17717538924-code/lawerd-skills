# 错误经验

## 已知异常模式

| 异常场景 | 根因 | 处置 |
|---------|------|------|
| 律师提供不完整材料，分析结论偏离 | 未在 step-01 硬Hook 中要求律师确认材料完整性 | 严格遵守硬Hook 规则，收到确认前不产出分析 |
| 跳过策略确认直接起草，被律师驳回 | step-02 硬Hook 未执行 | 不允许跳过 step-02，策略必须经律师确认 |
| DOCX 格式与模板不一致 | 模板填充时直接编辑了格式 XML | 仅替换 `<w:t>` 文本，不触碰格式 XML |
| 修改循环超过 3 轮后仍未定稿 | 争议点未在起草前确认 | 第 3 轮后提示面谈，不继续在线修改 |
| 归档到错误客户文件夹 | 律师未确认文件夹命名即执行归档 | 未找到匹配文件夹时必须触发硬Hook，不推测 |
