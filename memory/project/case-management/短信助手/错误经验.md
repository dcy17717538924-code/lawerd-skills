# 错误经验

## 已知异常模式

| 异常场景 | 根因 | 处置 |
|---------|------|------|
| 短信内容不完整（缺案号或无链接），court-sms 无法解析 | 律师只粘贴了部分短信内容 | 要求律师提供完整短信内容或手动补充案号 |
| 送达平台需要验证码，下载流程中断 | court-sms 不含验证码处理逻辑 | 提示律师手动下载后提供文件路径 |
| 归档后未询问是否更新案件进展日志 | court-sms 完成汇报后直接结束 | 遵循兜底规则：归档完成后必须询问"是否需要更新案件进展日志？" |
| 判决书下载后直接归档但错过了上诉期限提醒 | court-sms 未正确计算上诉截止日期 | 衔接动作的汇报模板中必须包含上诉截止日期 |
