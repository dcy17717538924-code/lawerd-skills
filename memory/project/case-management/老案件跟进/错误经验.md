# 错误经验

## 已知异常模式

| 异常场景 | 根因 | 处置 |
|---------|------|------|
| 未找到案件进展日志，agent 自行推测案情 | 兜底规则未执行，agent 跳过 Hook 直接分析 | 严格遵守兜底规则：日志不存在时在 Hook 中提示律师，不自行推测 |
| 新材料与历史分析结论矛盾时未提示律师 | step-01 硬Hook 中未要求确认理解偏差 | 硬Hook 必须包含"对当前案件阶段的理解是否有偏差" |
| 律师提供多份材料但只处理了其中一部分 | step-01 预处理时遗漏了文件 | 预处理完成后在软Hook 中列出所有已处理的文件，供律师确认完整性 |
