# step-02：分析新材

## 目的

基于律师对背景简报的反馈，产出调整后的诉讼策略或分析结果。

## 输入

- step-01 产出的案件背景简报
- 律师对硬Hook 的回复

## 输出

- 调整后的策略分析报告（含五要素）

## 前置条件

- step-01 的硬Hook 已完成
- 律师已回复确认事项

## 执行

1. 读取律师对 step-01 硬Hook 的回复
2. 产出调整后的分析结果，策略分析须包含：
   - 法律关系定性
   - 请求权基础
   - 举证责任分配
   - 风险识别
   - 建议策略及备选方案

## Hook

- **硬Hook**：询问律师——
  - 分析结果是否有需要调整的地方
  - 需要起草何种材料（诉状、申请书、辩论意见、证据目录等）
