# 错误经验

## 已知异常模式

| 异常场景 | 根因 | 处置 |
|---------|------|------|
| 入卷材料分类错误（诉讼 vs 非诉大类混用） | case-archive-orchestrator 自动分类规则与律所实际分类不一致 | 在案件目录写临时脚本来覆盖默认分类逻辑 |
| 生成案卷目录时模板字段缺失（如案号为空） | 模板中占位符未全部映射到案件信息 | 归档前确认所有必填字段已填充 |
| 命名规范与律所最新要求不符 | case-archive-orchestrator 中的命名规则未及时更新 | 使用案件目录临时脚本驱动全流程 |
