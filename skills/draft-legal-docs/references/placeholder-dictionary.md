# 法律文书通用占位符词典

> 版本：v1.1.0
>
> 定义所有法律文书中出现的信息变量占位符，按语义域分类。
> 模板入库：具体信息 → 占位符。起草填充：占位符 ← 案件事实。

---

## 符号约定

| 标记 | 含义 |
|------|------|
| `[变量名]` | 占位符，起草时按案件事实替换 |
| ⚡ | 必填，缺少时硬 Hook 追问 |
| ○ | 选填，有则填、无则删除该占位符所在行 |
| △ | 条件必填，视案件类型/诉讼地位决定 |

## 使用规则

### 模板匹配后
1. 加载对应 `.template.docx` 或 `.template.xlsx`
2. 提取模板中所有 `[占位符]`
3. 按本词典分类列出"需提供的案件信息清单"
4. ⚡ 必填项 → 硬 Hook 追问律师
5. ○ 选填项 → 有默认值的自动填，无默认值的跳过
6. △ 条件项 → 根据案件类型判断是否追问

### 起草填充后
仍有未填充的 ⚡ → 生成"缺失信息清单"附在初稿后
