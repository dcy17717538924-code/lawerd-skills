# step-05：知识沉淀

## 目的

定稿交付后，提示律师对比初稿与定稿差异，沉淀写作习惯。

## 输入

- 初稿文本
- 定稿文本

## 输出

- 写作习惯记录（写入 `D:\Agent知识库\写作习惯\`）

## 执行

1. 定稿交付后执行软Hook 提示
2. `IF` 律师确认 `THEN` 读取初稿与定稿差异 → 对比 → 写入 `D:\Agent知识库\写作习惯\`
3. `IF` 律师跳过 `THEN` 不阻塞，正常结束

## Hook

- **软Hook**：提示"是否需要对比初稿与定稿差异，沉淀到写作习惯？"
