# 办案实操指南

## 功能

不分析法律，只回答"怎么办"——去哪、带什么、什么流程、注意什么。

## 触发条件

以下任一模式触发：

| 模式 | 示例 |
|------|------|
| 问地点 | "调口卡去哪"、"工商内档在哪查"、"不动产登记中心" |
| 问材料 | "起诉要带什么材料"、"协执需要什么" |
| 问程序 | "怎么申请财产保全"、"公示催告流程" |
| 问机构 | "交诉讼费去哪个窗口"、"送达回证怎么弄" |

## 工作流

### 第一步：检索本地经验库

先从 `manage-legal-knowledge` 已整理的知识点中检索匹配内容。

### 第二步：联网补充

本地经验库未命中时，联网搜索最新政策和流程。

## 原则

- **精准优于泛泛**——优先经验库，不知道就如实说
- **时效性标注**——涉及政策的加日期标注
- **不替代官方渠道**——末尾加"以上仅供参考，以办事机构当场告知为准"
