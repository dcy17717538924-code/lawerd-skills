# 深度模式(6 阶段)

复杂争议、证据多、金额大的案件,分阶段逐步推进,每阶段停顿确认。

## 阶段流程

按顺序执行,每阶段停顿确认:

1. **事实梳理**:时间线 + 7W2H + 事实分层(有利/不利/争议/待查)
2. **法律分析**:法律/司法解释/地方规则分层检索
3. **法理研究**:立法目的、法律漏洞、解释路径
4. **案例检索**:指导案例优先,受案法院体系跟进
5. **裁判规则**:要件规则、免责例外、高影响因素
6. **结果预判**:三段论涵摄 + 政策修正 + 个案微调

## 执行原则

- 每完成一个阶段,**主动总结输出**
- 超长中断后**先复核再续**,避免前后矛盾
- 阶段之间给{{USER_SHORT_NAME}}确认节点,不跳步

## 跟简易模式的取舍

| 维度 | 简易模式 | 深度模式 |
|---|---|---|
| 适用 | 事实清晰、争议小、金额小 | 复杂、证据多、金额大 |
| 流程 | 三步 / 三轮 | 6 阶段 |
| 文档 | 案件分析报告(模板化) | 同上,但各阶段可独立沉淀 |
| 时间 | 单 session 可完成 | 多 session 协同 |
