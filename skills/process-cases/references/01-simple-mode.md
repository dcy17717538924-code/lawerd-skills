# 简易模式

事实清晰、争点明确,或仅有具体法律问题无完整材料的咨询。

## 入口问询

进入简易模式后,先问:"你想咨询一个法律问题,还是有案件需要处理?"

- **场景 A:轻分析**(仅具体法律问题,无完整案情材料)→ 三步轻分析流程
- **场景 B:快审**(有完整材料,事实清晰争议明确)→ 三轮快审流程

---

## 场景 A:轻分析(三步)

针对"这个问题怎么看"、"是否构成 XX"等具体法律问题。

### 第 1 步:确认问题与事实背景

- 复述法律问题,追问:"请提供与该问题相关的具体案件事实或背景情况。若无具体案件,请说明是纯理论探讨。"
- 概括提供的事实,请求确认:"您提供的分析背景是……,我的概括是否准确?请确认。"

### 第 2 步:结论先行 + 法律依据陈列

- 获得确认后,**首段直接给出核心分析结论**。
- 紧接着明确列出分析所依据的**法律规范名称及具体条号**。
- 如涉及地方法院口径,主动检索知识库中已有的专题知识。

### 第 3 步:逐步演绎与风险提示

- 结合已确认的事实,展开法律适用逻辑的分析。
- **必须包含"风险提示"部分**,指出论证弱点、不同解释可能性或证据缺失风险。
- 从对方律师视角,预测可能的攻击方向。

---

## 场景 B:快审(三轮)

### 第 1 轮:事实+证据快审

输出两张表:

1. **事实三分表**:有利事实 / 不利事实 / 争议事实
2. **要件-事实-证据对照表**:请求权基础 → 构成要件 → 对应事实 → 现有证据 → 状态(✅⚠️❌)

结尾提问:"以上是否有遗漏(尤其不利事实)?"

### 第 2 轮:法律+案例快检

- 法律/司法解释清单(法律全称+条号)
- 3-5 个相关案例(案号/法院/要旨/对本案影响)

结尾提问:"你是否同意我对法律关系的定性?"

### 第 3 轮:预判+行动

输出:

- 各项诉请支持度预判(高/中/低风险)
- 各项诉请预判表
- 3 条策略建议(证据补强/程序动作/谈判窗口)
- 下一步仅 2 项行动

结尾提问:"如果没有新的补充,我进入最终报告输出阶段。"
