# 检索与引用规则 + 禁止项

## 检索与引用规则

1. 法条必须标明**全称及具体条号**
2. 案例必须标注**案号/法院/要旨**
3. 无法核实处明确标注"**待核**"
4. 结论必须与证据状态联动:证据 ❌ 的要件不得给"高把握"
5. 法条核验走 IMA 国家法律知识库(凭证见主 agent memory `tech-environment.md`)
6. 案例检索走 IMA 人民法院案例库

## 禁止项(本 skill 专属)

- ❌ 不承诺"一定胜诉"
- ❌ 不跳步(简易模式三轮 / 深度模式六阶段必须按序)
- ❌ 不忽视对方抗辩路径
- ❌ 不脱离证据谈结论
- ❌ 不在聊天界面完整展开案件分析报告(只提示已生成)

## 红线(与主 agent 粽宝工作原则同步,不得违反)

- ❌ 信息缺失用"某"等占位符代替 —— 必须向{{USER_SHORT_NAME}}询问准确信息
- ❌ {{USER_SHORT_NAME}}未审阅前去掉"草案"标注
- ❌ 预测诉讼结果或使用情绪化/营销化用语
- ❌ 不暴露我方诉讼策略(事实与理由只说核心事实和法律依据)

- [ ] 法条是否标明全称+条号
- [ ] 案例是否标注案号/法院/要旨
- [ ] 不利事实是否完整呈现(尤其快审第 1 轮)
- [ ] 风险提示是否到位(尤其轻分析第 3 步)
