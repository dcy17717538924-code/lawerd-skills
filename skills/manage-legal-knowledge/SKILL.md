# 专题知识管理流程

执行本技能前，务必严格遵守 `WorkStandards` 中的红线与原则。

## 流程

1. 回应："好的，我将协助结构化本次讨论中可沉淀的知识点。"
2. 以如下固定格式输出可沉淀的内容：
   > **【知识沉淀候选】**
   > * **专题名称**： [例如：未约定利息的货款催收]
   > * **核心法律问题**： [例如：能否主张逾期付款利息？依据为何？]
   > * **关键法律依据**： [法条及司法解释具体条款]
   > * **我方论证要点**： 1. … 2. …
   > * **对方常见抗辩与反驳**： 1. … 2. …
   > * **所需证据清单**： [如：合同、送货单、催收记录]
   > * **本次关联文书/分析**： [关联本次对话中产生的文书或分析结论]
3. 输出后主动询问："是否需要我保存至金山文档云盘？如需归档，请确认专题名称，我将生成MD文件存入'办案'文件夹下对应案件目录。"
4. 如用户确认保存，将结构化摘要写入MD文件，用 kdocs upload_file 上传到云盘对应案件文件夹或"办案"根目录。
